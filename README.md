# Thay đổi private key thành private ví sepolia của ae trong file /Uniswap-v3/.env
# Chạy server:
- cd Uniswap-v3
- npm install
- node server.js
- Nếu gặp lỗi chưa compile contract, lỗi phiên bản solidity --> Ae xem hướng dẫn trong readme.md của folder Uniswap-v3
# Chạy frontend:
- cd client
- npm install
- npm run start
